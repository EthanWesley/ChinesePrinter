使用说明：
1. 把 Luckfox Pico KVM 通过 USB 连到目标电脑（KVM 会枚举为 HID 键盘）
2. 在目标电脑上把光标放到输入位置（记事本、浏览器地址栏等均可）
3. 确保目标电脑 NumLock 已开启（Alt 码必须用小键盘数字）
4. 在本页面输入中文，点击「开始打字」

编码原理：
GBK 模式：按住 Alt + 输入 GBK 十进制码 + 松开 Alt（如 "中"=54992）
Unicode 模式：按住 Alt + 输入 Unicode 十进制码点 + 松开 Alt（如 "中"=20013）
